# О проекте

Проект представляет собой веб-сервис по просмотру и редактированию задач. Проект был разработан по тестовому заданию Framework Team.

## Функционал

В данном проекте представлен следующий функционал:

- Регистрация пользователя;
- Авторизация пользователя;
- Просмотр задач;
- Создание задачи;
- Изменение статуса выполнения задачи;
- Удаление задачи.

## Технологии

- PHP 8.3;
- Composer;
- NodeJS и npm.

## Запуск локально

### 1. Настройка базы данных

В данном проекте используется база данных SQLite.
Выполните миграции для создания таблиц в базе данных:

```bash
php artisan migrate
```

### 2. Установка зависимостей

Для запуска проекта требуется установить зависимости PHP и JavaScript:

Для установки JavaScript-зависимостей выполните команду:

```bash
npm install
```

Для установки PHP-зависимостей выполните команду:

```bash
composer install
```

### 3. Запуск Frontend сервера

Для запуска локального Frontend-сервера откройте дополнительный терминал и выполните команду:

```bash
npm run dev
```

### 4. Запуск PHP сервера

Для запуска локального PHP-сервера выполните следующую команду:

```bash
php artisan serve
```

После выполнения команды в терминале появится следующая ссылка, ведущая на запущенный проект:

```bash
Server running on [http://127.0.0.1:8000]. 
```
